## 什么是科学上网？

通过技术手段访问被限制的国外网站和服务：

- 流媒体：Netflix、YouTube、Disney+
- 社交平台：Twitter、Instagram、Facebook
- AI服务：ChatGPT、Claude、Midjourney
- 开发工具：GitHub、Google、Stack Overflow

## 为什么选择 Clash？

| 特性 | 说明 |
|------|------|
| 规则分流 | 国内网站直连，国外网站代理 |
| 多协议支持 | Shadowsocks、V2Ray、Trojan |
| 界面友好 | 图形化操作，新手快速上手 |
| 跨平台 | Windows、macOS、iOS、Android |

## 常见问题

### Q: 打不开怎么办？

1. 以管理员身份运行
2. 检查防火墙设置
3. 删除 .config/clash 文件夹

### Q: 如何选择节点？

- 日常浏览：延迟低的节点
- 视频播放：带宽大的节点
- 游戏加速：专线节点
