# Netflix Unlock Detection

## Why Need Detection?

Netflix has different content strategies for different regions. Some nodes may:
- ❌ Completely inaccessible
- ⚠️ Only original content available
- ✅ Full access to all content

---

## Detection Methods

### Method 1: Netflux Website

Visit [Netflux](https://netflux.netlify.app/) to automatically test node streaming support.

### Method 2: Test Video

Visit Netflix test videos:
- [Test Video 1](https://www.netflix.com/watch/70143404)
- [Test Video 2](https://www.netflix.com/watch/80018499)

If can't play, the node doesn't support Netflix.

### Method 3: Netflix Official Website

Directly visit [Netflix.com](https://netflix.com):
- If prompted "Not Available", node not supported
- If normal access, test by playing videos

---

## Unlock Status Explained

| Status | Description |
|--------|-------------|
| Full | Full access to all content |
| Original | Original content only |
| None | Cannot access |

---

## How to Find Netflix-Supported Nodes?

1. **Provider Labels**: Most providers label Netflix-supported nodes
2. **Ask Support**: Inquire via TG group or ticket
3. **Self-Test**: Use above methods to test
