# Netflix 解锁检测

## 为什么需要检测？

Netflix对不同地区的IP有不同的内容策略，某些节点可能：
- ❌ 完全无法访问
- ⚠️ 只能访问原创内容
- ✅ 完整解锁所有内容

---

## 检测方法

### 方法1：Netflux 网站

访问 [Netflux](https://netflux.netlify.app/)，自动检测节点支持情况。

### 方法2：测试视频

访问 Netflix 测试视频：
- [测试视频1](https://www.netflix.com/watch/70143404)
- [测试视频2](https://www.netflix.com/watch/80018499)

如果无法播放，说明节点不支持Netflix。

### 方法3：Netflix官网

直接访问 [Netflix.com](https://netflix.com)：
- 如果提示"不可用"，节点不支持
- 如果正常访问，可以播放视频测试

---

## 解锁状态说明

| 状态 | 说明 |
|------|------|
| Full | 完整解锁所有内容 |
| Original | 仅原创内容 |
| None | 无法访问 |

---

## 如何找到支持Netflix的节点？

1. **机场标注**：大部分机场会标注支持Netflix的节点
2. **咨询客服**：在机场TG群或工单询问
3. **自行测试**：使用上述方法测试
